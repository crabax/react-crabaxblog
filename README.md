# react-crabaxblog

react-crabaxblog is a ReactJS client for the *crabaxblog-server* webapi.

## Installation

Use the package manager [npm](https://docs.npmjs.com/getting-started) to install dependencies.

```bash
npm install
```

## Run
```bash
npm start
```

## ESLint check
```bash
npm run lint:js
```
